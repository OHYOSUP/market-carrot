## 로그인

- Prsima Severless 데이터베이스로 User, Product, Chat, Community등의 model 생성
- Cloudflare Images를 이용해 효율적인 이미지 데이터 관리

- SWR로 api 데이터를 캐싱, 관리했고, 좋아요 기능과 채팅 기능은 Mutate
